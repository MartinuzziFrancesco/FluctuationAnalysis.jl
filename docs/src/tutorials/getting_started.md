# Getting started

This tutorial walks through a complete detrended fluctuation analysis (DFA)
workflow with [`dfa`](@ref) and introduces the options that every other method in
the package shares.

## A first analysis

DFA measures how the fluctuations of a signal grow with the size of the window
over which they are measured. The slope of that growth in log-log coordinates is
the *scaling exponent*: ``0.5`` for uncorrelated noise, larger for
positively-correlated signals.

```@example gs
using FluctuationAnalysis
using Random

series = randn(MersenneTwister(1), 10_000)   # uncorrelated white noise
result = dfa(series)
result
```

The scaling exponent is read with [`scaling_exponent`](@ref):

```@example gs
scaling_exponent(result)
```

As expected for white noise it is close to ``0.5``.

## The result object

[`dfa`](@ref) returns a [`DFAResult`](@ref) that keeps the full analysis, not just
the exponent:

```@example gs
result.scales         # the window sizes that were evaluated
```

```@example gs
result.fluctuations   # the fluctuation function F(s), one value per scale
```

```@example gs
result.fit            # the log-log fit
```

The [`LogLogFit`](@ref) also records the intercept, the coefficient of
determination, and the range of scales used:

```@example gs
result.fit.rsquared
```

## Choosing the scales

By default the scales are generated by [`logarithmic_scales`](@ref). You can pass
your own range, or tune the generated one:

```@example gs
scales = logarithmic_scales(length(series); minimum_scale=16, maximum_scale=1000)
result = dfa(series; scales=scales)
result.scales
```

Restrict the regression to a band of scales without changing which scales are
computed using `fitrange`:

```@example gs
dfa(series; scales=scales, fitrange=(32, 512)) |> scaling_exponent
```

## Choosing the detrender

The local trend removed from each segment is controlled by an
[`AbstractDetrender`](@ref). The default is a linear [`PolynomialDetrender`](@ref);
the `order` keyword is a shortcut for building a higher-order one:

```@example gs
dfa(series; order=2) |> scaling_exponent     # quadratic detrending
```

Equivalently, pass a detrender explicitly:

```@example gs
dfa(series; detrender=PolynomialDetrender(2)) |> scaling_exponent
```

## Detecting correlations

On a correlated signal the exponent rises above ``0.5``. A cumulative sum turns
white noise into a Brownian motion, whose DFA exponent is close to ``1.5``:

```@example gs
walk = cumsum(randn(MersenneTwister(2), 10_000))
scaling_exponent(dfa(walk))
```
